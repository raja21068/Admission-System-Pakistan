1. Using JAXB Binding Declarations 

Standard JAXB binding declarations are declared in the namespace, http://java.sun.com/xml/ns/jaxb/. The syntax of the binding declarations are specified by the JAXB binding schema. The JAXB binding schema contains a version attribute which indicates the version of the JAXB specification as well as the version of the schema itself.

The file name of the JAXB binding schema contains the specific version of the related specification. This is introduced for convenience to locate specific versions of the schema document. Schemas containing inlined JAXB binding declarations and external;bindings files are not required to refer to the specific version of the file. Instead, an XML schema document referencing the jaxb namespace refers to the logical JAXB namespace, http://java.sun.com/xml/ns/jaxb, and specifies the version of the binding schema as is described in the Customization Section of the JAXB specification.
